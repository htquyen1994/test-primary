## 1. FRAMEWORK SỬ DỤNG PATTERNS

```
Trước khi dùng bất kỳ pattern nào, hỏi:
  1. Xu hướng hiện tại là gì? (HH/HL hay LH/LL?)
  2. Pattern ở đâu? (Tại S/R? Giữa không khí?)
  3. Volume như thế nào? (Tăng hay giảm?)
  4. Pattern thuận hay ngược xu hướng?
  5. Stop loss ở đâu? Target ở đâu?
```

> **Nguyên tắc vàng:** Pattern tại S/R + thuận xu hướng + volume xác nhận = Xác suất cao nhất

## 5. WYCKOFF PATTERN — TÓM TẮT

### Chu kỳ thị trường:
```
Accumulation → Markup → Distribution → Markdown → (lặp lại)
```

### Wyckoff Accumulation (Bullish):
```
Phase A: SC (Selling Climax) + AR (Automatic Rally) + ST
Phase B: Trading range (tích lũy)
Phase C: Spring — Phá xuống GIẢ, bẫy bears
Phase D: SOS (Sign of Strength) — Bắt đầu markup
Phase E: Uptrend

→ Entry: Tại Spring hoặc khi SOS xác nhận
→ Dấu hiệu Spring: Phá support → Ngay lập tức đảo chiều mạnh
```

### Wyckoff Distribution (Bearish):
```
Phase A: BC (Buying Climax) + AR + ST
Phase B: Trading range (phân phối)
Phase C: UTAD — Phá lên GIẢ, bẫy bulls
Phase D: SOW (Sign of Weakness) — Bắt đầu markdown
Phase E: Downtrend

→ Entry: Tại UTAD hoặc khi SOW xác nhận
→ Dấu hiệu UTAD: Phá resistance → Ngay lập tức đảo chiều mạnh
```

## 6. CÁCH GIAO DỊCH THEO TỪNG NHÓM PATTERN

### Nhóm 1: Reversal tại S/R (Ưu tiên cao nhất)
```
Setup:
  1. Xác định S/R quan trọng
  2. Chờ giá tiếp cận S/R
  3. Tìm reversal pattern tại đó
  4. Volume giảm khi tiếp cận, tăng khi đảo chiều
  
Entry: Trên/dưới pattern
Stop: Dưới/trên S/R
Target: S/R tiếp theo
```

### Nhóm 2: Continuation sau Pullback
```
Setup:
  1. Xác định xu hướng chính
  2. Chờ pullback về S/R / MA / Fibonacci
  3. Tìm continuation pattern (3 Bar Play, Flag, Inside Bar)
  4. Volume giảm trong pullback
  
Entry: Breakout theo hướng xu hướng chính
Stop: Dưới/trên đáy/đỉnh pullback
Target: Measured Move hoặc S/R tiếp theo
```

### Nhóm 3: Breakout Patterns
```
Setup:
  1. Xác định pattern đang hình thành (Triangle, Rectangle, Flag)
  2. Chờ breakout với volume tăng
  3. Xác nhận sustenance (2-3 nến đóng cửa ngoài pattern)
  
Entry: Khi breakout xác nhận
Stop: Trong pattern (dưới/trên đường biên)
Target: Chiều cao pattern chiếu từ điểm breakout
```

```

  ✅ Volume GIẢM khi tiếp cận đỉnh/đáy thứ 2 (kiệt sức)
  ✅ Volume TĂNG khi breakout qua Neckline

Continuation Patterns:
  ✅ Volume LỚN trong Flag Pole / Impulse
  ✅ Volume GIẢM trong consolidation (Flag, Pennant, Rectangle)
  ✅ Volume TĂNG khi breakout

Wyckoff:
  ✅ Volume CAO tại SC/BC (climax)
  ✅ Volume THẤP trong trading range
  ✅ Volume TĂNG tại SOS/SOW
```

```

✅ Không bán ngay trên support lớn?
```

## 11. NGUYÊN TẮC VÀNG

1. **Vị trí > Hình dạng** — Cùng pattern, khác vị trí = khác ý nghĩa
2. **Volume xác nhận tất cả** — Breakout không có volume = nghi ngờ
3. **Pattern yếu cần xác nhận** — Harami, Hanging Man không vào ngay
4. **Gap hiếm trong intraday** — Island Reversal, Abandoned Baby ít gặp
5. **Wyckoff cần khung lớn** — Hiệu quả nhất trên Daily/Weekly
6. **Thất bại kỳ vọng = Tín hiệu mạnh** — Khi thị trường không làm điều bạn kỳ vọng
